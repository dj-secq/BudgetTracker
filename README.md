# BudgetTracker 💸
### Minimalism meets Financial Freedom.

BudgetTracker is a clean, modern Android application built to help users take control of their finances using the **50/30/20 rule**. With an immersive edge-to-edge design and an intuitive user interface, managing your money has never been this satisfying.

---

## ✨ Key Features

- **🏠 Smart Dashboard**: View your total balance, monthly income, and expenses at a glance in a clean, centered interface.
- **📈 50/30/20 Rule Insights**: Automatically track how your spending aligns with your **Needs**, **Wants**, and **Savings** targets.
- **🎯 Savings Goals**: Create and fund individual savings goals with progress tracking and direct wallet integration.
- **🕒 Detailed Transactions**: A grouped history of your activity with powerful filtering by category, account, and date range.
- **⚠️ Budget Warnings**: Get real-time alerts if a transaction exceeds your monthly category budget or life-stage allocation.
- **🎨 Immersive UI**: Fully edge-to-edge design with support for **Light**, **Dark**, and **System** themes.
- **💳 Multi-Wallet Management**: Manage multiple accounts/wallets with custom colors and types.
- **📁 Categorization**: 20+ default categories with smart icon mapping for quick and easy entries.

---

## 🛠 Tech Stack

- **Language**: [Kotlin](https://kotlinlang.org/)
- **UI Framework**: [Jetpack Compose](https://developer.android.com/jetpack/compose) (100% declarative UI)
- **Architecture**: MVVM (Model-View-ViewModel)
- **Local Database**: [Room Persistence Library](https://developer.android.com/training/data-storage/room)
- **Preferences**: [DataStore](https://developer.android.com/topic/libraries/architecture/datastore)
- **Navigation**: [Compose Navigation](https://developer.android.com/jetpack/compose/navigation) with custom slide transitions.
- **Dependency Injection**: Manual injection (Constructor-based) for simplicity and performance.

## 🚀 Getting Started

### Prerequisites
- Android Studio Ladybug (2024.2.1) or newer.
- Android SDK 24 (Nougat) or higher.

### Installation
1. **Clone the repository**:
   ```bash
   git clone https://github.com/YOUR_USERNAME/BudgetTracker.git
   ```
2. **Open in Android Studio**:
   - Select `File > Open` and navigate to the cloned project folder.
3. **Build and Run**:
   - Wait for Gradle sync to complete.
   - Click the **Run** button (green play icon) to deploy to an emulator or physical device.
